## Structural Design Patterns Quiz

Answer the questions first, then check the key.

## Questions
1. Which pattern adapts one interface to another?
2. Which pattern wraps objects to add behavior?
3. Which pattern provides a simplified interface?
4. Which pattern composes objects into trees?
5. Which pattern controls access to an object?

## Answer key
1. Adapter.
2. Decorator.
3. Facade.
4. Composite.
5. Proxy.
