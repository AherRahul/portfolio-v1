## Behavioral Design Patterns Quiz

Answer the questions first, then check the key.

## Questions
1. Which pattern defines a family of algorithms?
2. Which pattern lets objects subscribe to events?
3. Which pattern encapsulates a request?
4. Which pattern allows stepping through a collection?
5. Which pattern changes behavior based on state?

## Answer key
1. Strategy.
2. Observer.
3. Command.
4. Iterator.
5. State.
