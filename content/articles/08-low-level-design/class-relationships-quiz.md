## Class Relationships Quiz

Answer the questions first, then check the key.

## Questions
1. Which relationship is strongest: aggregation or composition?
2. What is association?
3. What is dependency?
4. When do you use composition?
5. What is realization?
6. Give an example of aggregation.

## Answer key
1. Composition.
2. A general uses-a relationship.
3. A temporary usage of another class.
4. When lifecycles are tightly bound.
5. A class implements an interface.
6. Department has Professors.
