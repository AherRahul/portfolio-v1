## Creational Design Patterns Quiz

Answer the questions first, then check the key.

## Questions
1. Which pattern ensures a single instance?
2. Which pattern builds complex objects step by step?
3. Which pattern creates families of related objects?
4. Which pattern clones objects?
5. What is the intent of Factory Method?

## Answer key
1. Singleton.
2. Builder.
3. Abstract Factory.
4. Prototype.
5. Delegate object creation to subclasses.
