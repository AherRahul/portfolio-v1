# git log

`git log` is the primary tool for exploring your project history in Git. It shows the commit history and is highly configurable with many options for formatting, filtering, and visualizing the history.


## Basic Usage

```bash
# Show full commit history
git log

# Default output per commit:
# commit abc1234def5678...
# Author: Jane Doe <jane@example.com>
# Date:   Mon Jan 1 2024
#
#     Add user authentication
```


## Formatting Options

### Compact Views

```bash
# One line per commit
git log --oneline
# abc1234 Add user authentication
# def5678 Fix login bug
# g9h0i1j Initial commit

# Short format (still compact but with more detail)
git log --format=short
```

### Custom Format

```bash
# Custom pretty format
git log --pretty=format:"%h %an %ar %s"
# %h  = abbreviated commit hash
# %H  = full commit hash
# %an = author name
# %ae = author email
# %ar = relative date (e.g., "2 days ago")
# %ad = absolute date
# %s  = subject (first line of commit message)
# %b  = body
# %d  = ref names (branches, tags)

# Colorized output
git log --pretty=format:"%Cred%h%Creset %Cblue%an%Creset %Cgreen%ar%Creset - %s"
```

### Showing Statistics

```bash
# Show changed files and line counts
git log --stat

# Show the full diff for each commit
git log -p

# Show only the file names that changed
git log --name-only

# Show file names with change type (A=Added, M=Modified, D=Deleted)
git log --name-status
```


## Filtering History

### By Number of Commits

```bash
git log -5             # Last 5 commits
git log -n 10          # Last 10 commits
git log --max-count=5  # Same as -5
```

### By Date

```bash
git log --since="1 week ago"
git log --until="2024-01-01"
git log --after="2024-01-01" --before="2024-02-01"
git log --since="yesterday"
```

### By Author

```bash
git log --author="Jane"
git log --author="jane@example.com"
```

### By Commit Message

```bash
git log --grep="authentication"   # Commits mentioning "authentication"
git log --grep="feat" --grep="fix" --all-match  # Must match all patterns
```

### By File

```bash
# Show commits that touched a specific file
git log -- src/auth.js

# Show commits that touched any file in a directory
git log -- src/

# Show commits that affected specific function or code
git log -S "authMiddleware"   # Commits that added or removed this string
git log -G "auth.*middleware" # Commits matching this regex
```


## Graph Views

```bash
# Show branch graph
git log --graph --oneline --all

# More complete graph with decorations
git log --graph --oneline --all --decorate

# Full view with author and date
git log --graph --pretty=format:"%Cred%h%Creset -%C(yellow)%d%Creset %s %Cgreen(%cr) %C(bold blue)<%an>%Creset" --all
```


## Comparing Ranges

```bash
# Commits in main but not in feature
git log feature..main
# or
git log main ^feature

# Commits in either branch but not both
git log main...feature

# Commits after a specific commit
git log abc1234..HEAD

# Commits in a branch since branching from main
git log main..feature
```


## Searching History

```bash
# Find who committed a specific string
git log -S "secretKey" --all

# Find commits that changed a function
git log -L :myFunction:src/app.js

# Find the commit a file was deleted in
git log --all --full-history -- deleted-file.js

# Find commits by tag range
git log v1.0.0..v2.0.0 --oneline
```


## Useful Aliases for Logging

```bash
# Add these to ~/.gitconfig [alias]
git config --global alias.lg "log --oneline --graph --all --decorate"
git config --global alias.history "log --pretty=format:'%h %ad | %s%d [%an]' --graph --date=short"
```
