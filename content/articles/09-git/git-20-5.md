# Git Tips and Tricks

A collection of powerful, lesser-known Git techniques that can dramatically improve your daily workflow.

## 1. Find What Branch a Commit is On

```bash
git branch --contains abc1234
git branch -r --contains abc1234  # Check remote branches too
```

## 2. List Files Changed in Last Commit

```bash
git diff --name-only HEAD~1 HEAD
git show --name-only  # Also shows commit message
```

## 3. Show All Unpushed Commits

```bash
git log origin/main..HEAD --oneline
# or
git cherry -v  # Also shows if commits are already upstream
```

## 4. Count Commits on a Branch

```bash
git rev-list --count HEAD  # Total commits
git rev-list --count main..feature  # Commits on feature not in main
```

## 5. Interactive Add Parts of a File

```bash
git add -p  # Stage specific hunks interactively
git checkout -p  # Discard specific hunks interactively
git reset -p  # Unstage specific hunks interactively
```

## 6. Undo a Pushed Commit Safely

```bash
# Revert creates a new commit that undoes the bad commit
git revert abc1234
git push origin main
```

## 7. Clone a Single Branch (Faster)

```bash
git clone --single-branch --branch main https://github.com/user/repo.git
```

## 8. See Which Commits Introduced/Removed a Function

```bash
git log -S "functionName" --patches
git log -G "functionName.*signature" --patches
```

## 9. Stage Files that Match a Pattern

```bash
git add '*.ts'        # Stage all TypeScript files
git add 'src/**/*.js' # Stage JS files in src/ recursively
```

## 10. Temporarily Ignore Tracked Files

```bash
# Tell git to ignore changes to this file locally
git update-index --assume-unchanged config/local.json

# Undo it
git update-index --no-assume-unchanged config/local.json

# List assumed-unchanged files
git ls-files -v | grep '^h'
```

## 11. Auto Squash Commits During Rebase

```bash
# Mark commits for automatic squashing
git commit --fixup=abc1234  # Will be squashed into abc1234
git commit --squash=abc1234  # With edit of combined message

# Then rebase with autosquash
git rebase -i --autosquash abc1234~
```

## 12. Find the Merge Base (Common Ancestor)

```bash
git merge-base main feature/login
# Returns: def5678...
# This is the common ancestor of both branches
```

## 13. Restore a Deleted File

```bash
# Find the commit that deleted it
git log --all --full-history -- lost-file.js

# Restore from the commit before deletion
git checkout <hash>^ -- lost-file.js
```

## 14. Show All Branches with Last Commit

```bash
git branch -vv         # Local branches with tracking info
git branch -r --sort=-committerdate | head -20  # Most recently active remotes
```

## 15. Quickly Switch to Previous Branch

```bash
git switch -   # Switch to the previously checked out branch
```

## 16. Print Current Branch Name

```bash
git branch --show-current
git rev-parse --abbrev-ref HEAD
```

## 17. View File at Different Commit Without Checking Out

```bash
git show HEAD~5:src/auth.js        # View file 5 commits ago
git show v1.0.0:README.md          # View file at specific tag
git show abc1234:package.json      # View file at specific commit
```

## 18. Edit Multiple Commits (Fixup Workflow)

```bash
# During development, commit fixups
git commit -m "feat: add login"
# ... later, realize you need to fix something in login ...
git add src/login.js
git commit --fixup HEAD~2  # Create fixup commit for 2 commits back

# At the end, rebase to clean up
git rebase -i --autosquash main
```

## 19. Find Large Files in History

```bash
git rev-list --objects --all | \
  git cat-file --batch-check='%(objecttype) %(objectname) %(objectsize) %(rest)' | \
  grep "^blob" | sort -k3 -rn | head -10
```

## 20. git Notes (Add Metadata to Commits)

```bash
# Add a note to a commit (doesn't change hash)
git notes add -m "Reviewed by Jane" abc1234

# Show notes in log
git log --show-notes

# Notes can be pushed/fetched separately
git push origin refs/notes/*
```
